# 最近お話

最近は日本関連のメディアコンテンツをよく見ています. そして、一番いやな時は, 一番盛り上がてるときに、しらない漢字が表示させるときです. 没入感速攻でが落ちた。
もう一つの大きな問題は、ブラウザが<strong>ふりがな</strong>をサポートしていないことです! しょんぼりぃー。 <br></br>
新しいタブを開いて漢字の意味を検索しなければならないことがよくあります。
これは、特に日本のフォーラムにいる場合は、すべての返信、すべてのディスカッション、コメント欄をコピーパーして必要があるため、シンプルに<strong>めんどくさい</strong>です。

彼は、私に<Link href="https://yomitan.wiki/" className="text-sky-400">ヨミタン</Link>という素晴しい機能を紹介しました。
ヨミタンはブラウザの拡張機能です。漢字の上にマウスを移動してShiftキーを押すだけで、その漢字の読み方や意味などがポップアウトで表示される。そっこうに、秒で。<br></br>
<strong>辞書はローカルに保存されているので、読み込み時間はあまりかかりません。</strong>
Chromiumベースのブラウザで問題なく機能しています。FirefoxやMicrosoft Edgeもサポートしています!

# ヨミタンの設定ガイド

ヨミタンは、10万人以上の言語学習者に利用されている、パワフルで多機能な言語学習用ポップアップ辞書です。設定も非常に簡単です、すぐに使えることができます。
自分もヨミタンを使いたいの方は、下に書かれている手順をみて、フォローし、一緒に設定しましょう：

1. Chromiumベースのブラウザを使っている方は: <Link href="https://chrome.google.com/webstore/detail/yomichan/" className="text-sky-400">Chrome Web Store</Link> からインストールしてください、または
   - Firefoxはこちらからインストールしてください: <Link href="https://addons.mozilla.org/en-US/firefox/addon/yomitan/" className="text-sky-400">Firefox Add-ons</Link>.
   - 最後にMicrosoft Edgeはこっち: <Link href="https://microsoftedge.microsoft.com/addons/detail/yomitan-popup-dictionary/idelnfbbmikgfiejhgmddlbkfgiifnnn" className="text-sky-400">Microsoft Edge Add-ons</Link>.
2. インストールが完成した後、 ヨミタンのアイコンがブラウザのツールバーに表示せれているはず
    <ImageOnly src="/images/posts/introduction-to-yomitan/yomitan-icon128.png"
      width="20"
      height="20"
      alt="Yomitan Icon"
      rounded="none" 
    /> そうでない場合は、「拡張機能」のボタンをクリックし確認してください。
3. そのアイコンをクリックして設定画面を開きましょう。
4. 設定画面で, ポップアップ辞書の外観と動作をカスタマイズできます。
5. 「辞書」タブをクリックして使用したい辞書を選択することで、ヨミタンに辞書を追加することもできます。私は個人的に <strong>Jitendex</strong> を使用しています。これは毎週新しいコンテンツで更新されます。<strong>もしJitendexがあなたにとって役に立つと思ったなら、もしよかったらプロジェクトに <ImageOnly alt="star" src="/images/star.png" width="20" height="20"/> を付けていただけると幸いです、こちらが <Link href="https://github.com/stephenmk/Jitendex" className="text-sky-400"> GitHub </Link> のリンクです。開発者に <Link href="https://ko-fi.com/jitendex" className="text-sky-400">Ko-fi</Link> をおごることもできます。</strong>
6. 辞書を追加おわったら Web サイトで漢字の上にマウスを移動して Shift キーを押すだけで、Yomitan を使い始めることができます。
7. ポップアップが表示され、読み方、ふりがな、意味、漢字の例がいくつか表示されます。

# 自分でもやってみてください

こちらにいくつか引用文があるので、マウスを<strong>ホーバー</strong>してポップアップ辞書を開いてみよう。漢字にマウスをホーバーして <strong>Shift</strong> キーを押すと、読み方と意味が表示されます。<br></br>
